Introduction

This webpage consists of the five elements created using BabylonJS for my JavaScript course at University. Please use the navigation bar at the top to explore the five elements and the documentation of them.
Bootstap was used to create the index page; allowing for a faster development of the page and the features implemented.

The Elements

The five elements that can be found on this webpage are created to showcase the following:

  • Element 1: Basic shapes, lighting, textures, shadows, and motion.
  • Element 2: A 3D environment placing a selection of individual and cloned textured meshes within a 360-degree scene with background image, textured terrain using a height map, appropriate lighting, and constrained camera motion.
  • Element 3: Movable player mesh incorporating player functional animations. Interaction between player and other objects using physics.
  • Element 4: A Graphic user interface featuring a menu system and transitions from menu to an interactive scene. At least one GUI element should change in response to scene events.
  • Element 5: Demonstration of at least two scenes which can be switched between either by GUI or in response to scene events. Each scene should incorporate the techniques explored in the module.
